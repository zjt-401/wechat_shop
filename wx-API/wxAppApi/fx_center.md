##  获取我的分销信息  ( /fx_center.html )
###  /fx_center.html  请求参数

|名称|说明|是否必要|备注
|:---:|:---:|:---:|:---:|
|canFx|有没有分销资格|是|-
|jsonOnly|返回json|是|-

|名称|说明|备注

|headimg|头像|-
|nickname|昵称|-
|parentId|父级id|-
|fxName|分销名|-
|fxTuijianma|推广号|-
|threeFxUserCount|三级分销总人数|-
|levelOneFxUserCount|1级子集人数|-
|levelTwoFxUserCount|2级子集人数|-
